# Micro Rush — Company Charter

## Mission
Build the future of personal computing: a privacy-first, proactive AI companion that is truly yours.

## Vision
Unlike cloud chatbots that forget who you are, Micro Rush lives on your hardware, monitors your world, and grows with you over months and years.

---

## Company Purpose
Micro Rush is a privacy-first, personal AI-assistant programming company. Our mission is to build an AI companion that is truly yours:
- **Privacy-First**: Your data stays on your hardware via local LLMs (Ollama)
- **Proactive**: Monitors calendar, messages, health — takes initiative
- **Persistent Memory**: Tiered memory system that remembers preferences and history

---

## The Micro Rush Engine
Stateful, agentic system powered by LangGraph.

### Core Components
| Component | Name | Description |
|-----------|------|-------------|
| The Brain | Tiered Memory | SQL (Recall) + Vector DB/LanceDB (Archival) |
| The Body | Plugin System | Dynamic registry for runtime skill learning |
| The Face | Web UI | Dark-mode React interface via FastAPI bridge |
| The Skills | Plugins | Calendar, File Management, Communication, Health, Finance |
